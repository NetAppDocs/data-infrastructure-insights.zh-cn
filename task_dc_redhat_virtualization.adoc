= Red Hat 虚拟化数据收集器
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
Red Hat Virtualization 的 DII 数据收集器还支持开源项目 oVirt 和 Oracle Linux Virtualization Manager。此数据收集器通过 REST API 与管理站上的 HTTPS 接口进行通信。



== Terminology

对于Data Infrastructure Insights获取的每种资产类型，都会显示该资产最常用的术语。查看或排除此数据收集器的故障时，请记住以下术语：

[cols="2*"]
|===
| 供应商/型号术语 | Data Infrastructure Insights术语 


| 磁盘 | 虚拟磁盘 


| 主机 | 主机 


| 虚拟机 | 虚拟机 


| 存储域 | 数据存储 


| 逻辑单元 | LUN 
|===
注意：这些只是常见的术语映射，可能并不代表此数据收集器的所有情况。



== 要求

* 通过 REST API 在端口 443 上访问 RHV/oVirt/OLVM 服务器的 IP 地址/主机名
* 只读用户名和密码
* RHV 版本 3.0+/Ovirt 4.0+/OLVM 4.0+




== 配置

[cols="2*"]
|===
| 字段 | 描述 


| RHV/Ovirt/OLVM 服务器 IP 地址 | 管理站的 IP 地址 


| 用户名 | 用户名 


| 密码 | 密码 
|===


== 高级配置

[cols="2*"]
|===
| 字段 | 描述 


| HTTPS 通信端口 | 用于与 RHEV 进行 HTTPS 通信的端口 


| 库存轮询间隔（分钟） | 默认值为 20 分钟。 


| VM 性能 | 启用性能收集 


| 性能轮询间隔（秒） | 默认值为 900 
|===
