= 节点数据收集
:hardbreaks:
:allow-uri-read: 


[role="lead"]
Data Infrastructure Insights从安装代理的节点收集指标。



== 安装

. 从*可观察性 > 收集器*中选择一个操作系统/平台。请注意，安装任何集成数据收集器（Kubernetes、Docker、Apache 等）也将配置节点数据收集。
. 按照说明配置代理。说明根据您用于收集数据的操作系统或平台的类型而有所不同。
