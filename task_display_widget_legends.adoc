= 显示小部件图例
:hardbreaks:
:allow-uri-read: 


[role="lead"]
仪表板上的窗口小部件可以显示图例，也可以不显示。

显示仪表板时，可以通过单击小部件上的“*选项*”按钮并在菜单中选择“*显示图例*”来打开或关闭小部件中的图例。

随着小部件中显示的数据发生变化，该小部件的图例也会动态更新。

当显示图例时，如果可以导航到图例指示的资产的登录页面，则图例将显示为指向该资产页面的链接。
