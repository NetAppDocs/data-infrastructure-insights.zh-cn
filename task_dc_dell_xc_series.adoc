= Dell EMC XC系列数据收集器
:hardbreaks:
:allow-uri-read: 


[role="lead"]
Data Infrastructure Insights使用此数据收集器来发现 Dell EMC XC 系列存储阵列的库存和性能信息。



== 配置

[cols="2*"]
|===
| 字段 | 描述 


| Prism 外部 IP 地址 | XC服务器的IP地址 


| 用户名 | XC 服务器的用户名 


| 密码 | XC服务器使用的密码 
|===


== 高级配置

[cols="2*"]
|===
| 字段 | 描述 


| TCP 端口 | 与 XC 服务器进行 TCP 通信所用的端口 


| 库存轮询间隔（分钟） | 库存调查之间的间隔。默认时间间隔为 60 分钟。 


| 性能轮询间隔（分钟） | 性能轮询之间的间隔。默认值为 300 秒。 
|===
