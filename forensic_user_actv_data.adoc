= 取证用户活动数据
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
用户活动页面可帮助您了解用户对实体执行的操作。

本页提供的数据显示：

* 实体访问时间，包括月、日、年、时间。
* 通过链接访问实体的用户link:forensic_user_overview.html["用户详细数据"]。
* 用户执行的活动：
+
** 写入
** 读取
** 访问元数据


* 指向实体的路径，其中包含指向link:forensic_entity_detail.html["实体详细数据"]
* 实体类型：
+
** 文件
** 路径


* 执行操作的 IP 地址。
