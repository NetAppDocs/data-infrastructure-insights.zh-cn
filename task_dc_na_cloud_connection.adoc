= 适用于 ONTAP 9.9+ 的 NetApp 云连接数据收集器
:hardbreaks:
:allow-uri-read: 


[role="lead"]
此数据收集器可创建云连接，以支持从 ONTAP 9.9+ CVO ， AFF 和 FAS 系统收集数据。


NOTE: 自2023年4月4日起、此数据收集器不再可安装在Data Infrastructure Insight中、并且将从2023年7月的所有Data Infrastructure Insight安装中删除。
