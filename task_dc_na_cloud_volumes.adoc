[role="lead"]
此数据收集器支持从适用于 AWS 的 NetApp Cloud Volumes Services 配置中收集清单。



== 配置

[cols="2*"]
|===
| 字段 | 说明 


| Cloud Volumes 区域 | 适用于 AWS 的 NetApp Cloud Volumes Services 所在地区 


| API 密钥 | Cloud Volumes API 密钥 


| 机密密钥 | Cloud Volumes 密钥 
|===


== 高级配置

[cols="2*"]
|===
| 字段 | 说明 


| 清单轮询间隔（分钟） | 默认值为60分钟 
|===
